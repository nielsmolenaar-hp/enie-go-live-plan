New here? The 3 tools you need to check any ticket (2-minute read)

Every checklist on this page uses at most three simple browser tools. You never need to understand code — you only open things, search for words, and compare what you see with what the checklist says.

Tool A — Read the page source

This shows the raw text Google receives. Open the page, right-click anywhere → View Page Source (a tab full of code opens — that's normal). Press Cmd+F (Mac) or Ctrl+F (Windows), type the search word from the checklist, and count the matches. That's all.

Tool B — Watch what the page downloads

Right-click the page → Inspect → click the Network tab at the top of the panel → reload the page. You'll see a list of everything the page fetches. Type a word in the panel's filter box to see if something with that name was downloaded.

Tool C — Run a speed test (Lighthouse)

Right-click the page → Inspect → click the Lighthouse tab → choose Mobile + Performance → click Analyze page load. Wait about a minute; it gives a 0–100 score plus named findings (the checklist tells you which finding to look for).

Extra — View the page as a phone

In the Inspect panel, click the small phone/tablet icon (top-left of the panel). The page reloads in a phone-sized frame.
